# 🛒 Advanced Exploratory Data Analysis on Superstore Sales Data

## 📌 Project Overview
This project performs Advanced Exploratory Data Analysis (EDA) on Superstore sales data using Python. The objective of this project is to analyze sales performance, identify trends and patterns, detect anomalies, validate assumptions using hypothesis testing, and generate meaningful business insights from the dataset.

This project was completed as part of the CodeAlpha Data Analytics Internship.

## 🎯 Objectives
- Understand the structure of the dataset
- Perform statistical analysis
- Identify sales and profit trends
- Detect outliers and anomalies
- Analyze relationships between variables
- Perform hypothesis testing
- Generate business insights for decision-making

## 🧠 Hypothesis Testing

### Null Hypothesis (H₀)
Technology and Furniture categories have equal average profit.

### Alternative Hypothesis (H₁)
Technology category has significantly different profit than Furniture category.
A T-Test was performed using SciPy to validate the hypothesis statistically.
